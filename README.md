# Monitoring Dashboard with Firebase Authentication

Dashboard monitoring real-time dengan sistem login menggunakan Firebase Authentication.

## Fitur
- ✅ Login dengan Email & Password
- ✅ Registrasi akun baru
- ✅ Role-based access (Admin/User)
- ✅ Monitoring real-time (Tegangan, Arus, Daya, Frekuensi)
- ✅ Kontrol pompa jarak jauh
- ✅ Grafik daya real-time
- ✅ Histori monitoring
- ✅ Informasi biaya (per jam, hari, bulan, total)

## Cara Deploy ke GitHub Pages

1. **Buat repository baru** di GitHub
2. **Upload semua file** berikut:
   - `index.html`
   - `dashboard.html`
   - `404.html`
   - `README.md`
3. **Aktifkan GitHub Pages**:
   - Buka Settings > Pages
   - Pilih branch `main` sebagai source
   - Klik Save

## Cara Menggunakan

1. Buka website GitHub Pages Anda
2. Register akun baru
3. Login dengan akun yang sudah dibuat
4. Akses dashboard monitoring
